用户命令¶

hai-cli init¶

初始化用户账户

Usage:

hai-cli init <token> [OPTIONS]

Options:

  • --url <url>

    required。初始化时，用户指定萤火二号的server，如果没特别需求不用填

    默认值: http://api.yinghuo.high-flyer.cn

  • --bff_url <bff_url>

    required。中转BFF url，如果没特别需求不用填。在萤火上会自动读环境变量，不用填；在萤火外，需要自己指定成绝对路径

    默认值: https://bff.yinghuo.high-flyer.cn/proxy/s

Arguments

  • token

    用于认证 hai 服务的密钥，由管理员分发

hai-cli whoami¶

显示用户的个人信息, 包括集群用量、quota 等

Usage:

hai-cli whoami [OPTIONS]

Options:

  • -a, --ascii

    使用 ASCII 字符打印制表符, 格式错乱时请开启此选项
